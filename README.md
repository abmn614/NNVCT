# Neural Network Visualization Builder

一个交互式神经网络可视化构建工具，帮助用户直观地理解和构建神经网络结构。

## 功能特性

### 网络结构配置
- **输入层/输出层**：支持自定义节点数量
- **隐藏层组（z/a）**：添加成对的隐藏层，模拟前向传播中的线性变换(z)和激活(a)
- **单层隐藏层**：灵活添加独立的隐藏层
- **动态增删**：实时添加或删除隐藏层

### 可视化视图
- **Canvas 绘制**：使用 HTML5 Canvas 绘制神经网络拓扑图
- **节点连接**：自动绘制层与层之间的连接线
- **权重显示**：在连接线上显示权重参数 w（支持上下标）
- **参数标签**：节点上方/下方显示自定义参数

### 参数编辑
- **权重参数(w)**：可在连接线上直接编辑权重值
- **节点参数**：支持自定义参数，包含：
  - 基础名（如 b、y）
  - 上标（如层编号）
  - 下标（如节点编号）
  - 顶标（帽子符号 ^）
  - 参数值
- **实时联动**：参数弹窗、参数标签、悬浮面板三处编辑实时同步

### 交互功能
- **画布拖拽**：支持拖拽画布查看完整网络
- **节点悬浮**：悬浮节点显示参数面板
- **面板固定**：可固定参数面板，支持拖拽移动
- **面板折叠**：支持折叠/展开参数面板
- **显示/隐藏参数**：一键切换所有参数标签的显示状态

## 快速开始

### 在线使用
直接在浏览器中打开 HTML 文件即可使用，无需安装任何依赖。

### 本地运行
```bash
# 克隆项目
git clone https://github.com/abmn614/NNVCT.git

# 进入目录
cd NNVCT

# 使用任意方式打开 HTML 文件
open index.html
# 或使用本地服务器
python -m http.server 8080
```

## 使用说明

### 1. 配置网络结构
- 在「各层节点配置」中调整输入层和输出层的节点数
- 点击「新增隐藏层组」添加 z/a 成对隐藏层
- 点击「新增单层隐藏层」添加独立隐藏层

### 2. 编辑参数
- **编辑权重**：直接在连接线上的输入框中修改权重值
- **编辑节点参数**：点击节点打开参数弹窗，可添加/修改/删除自定义参数
- **快捷编辑**：在参数标签或悬浮面板中直接修改参数值

### 3. 查看网络
- 拖拽画布查看完整网络结构
- 悬浮节点查看参数详情
- 点击固定图标锁定参数面板

## 技术栈

- **HTML5 Canvas**：网络拓扑绘制
- **Tailwind CSS**：UI 样式框架
- **原生 JavaScript**：交互逻辑
